# 世博轮腿机器人 - 稳定畅玩版 (Stable & Ready-to-Play Edition)

此项目是世博同学轮足机器人的 Arduino 工程转换而成的 PlatformIO 工程。
可以使用 VSCode 进行编译下载，彻底解决了 Arduino 编译巨慢的问题。

* **世博同学轮足机器人原项目地址**：[https://github.com/MuShibo/Micro-Wheeled_leg-Robot](https://github.com/MuShibo/Micro-Wheeled_leg-Robot)
* **世博开源轮足机器人TOMMY化版项目地址**：[https://gitee.com/myperworld/wrobot](https://gitee.com/myperworld/wrobot)

## ✨ 本版核心优化 (稳定畅玩版)

我们在前人开源代码的基础上，针对新手复刻最容易卡壳的问题进行了修复，主打**及烧即用**：

* 🎮 **解决手柄连接匹配痛点**
  原版代码中存在遥控器 MAC 地址硬编码绑定的问题，而“盖世小鸡一代”等手柄每次重启时 MAC 地址都会改变，导致小车与手柄始终连不上。本项目修改了匹配逻辑，消除了 MAC 地址强绑定的限制。（*注：目前仅针对“盖世小鸡一代”手柄进行了流畅连接实验，Xbox 理论支持*）。
* ⚙️ **修复硬件误差与控制参数**
  彻底解决了舵机零点俯仰角倾斜的问题；重新调优了部分 PID 参数，大幅降低了炸机概率，新手小白也能一次点亮。
